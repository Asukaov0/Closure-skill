# Contributing / 贡献指南

感谢你愿意一起改进 Closure Skill。这个项目欢迎更好的编程协作体验、更稳定的角色一致性、更清晰的文档，以及更安全的开源分发方式。

Thanks for helping improve Closure Skill. Contributions are welcome when they improve programming usefulness, persona consistency, documentation clarity, or redistribution safety.

## Guidelines / 贡献准则

- 优先保证它对真实编程工作有用。 / Keep the skill useful for real programming work first.
- 保留第一人称可露希尔身份契约。 / Preserve the first-person Closure identity contract.
- 可以有明日方舟社区味，但不要把 skill 写成设定百科。 / Keep it fan-friendly, but do not turn the skill into a lore encyclopedia.
- 不要加入官方美术、提取语音、长篇官方台词、游戏数据 dump 或专有文本。 / Do not add official artwork, extracted voice lines, long official quotes, game data dumps, or proprietary source text.
- 示例必须原创、简短，并适合公开再分发。 / Keep examples original, short, and safe for redistribution.
- 优先使用简洁、可验证的 skill 指令。 / Prefer concise, testable skill instructions.
- 提交前校验 `SKILL.md`。 / Validate `SKILL.md` before submitting changes.

## Validation / 校验

运行 / Run:

```powershell
python -X utf8 path/to/quick_validate.py .
```

期望结果 / Expected result:

```text
Skill is valid!
```
